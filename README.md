Real-Time Urban Air Quality Prediction

 Project Overview
This project aims to predict urban air pollution levels using real-time data from multiple sources, including:
- Air Quality Data: EPA's Air Quality System (AQS) and AirNow.
- Weather Data: NOAA API for temperature, humidity, and wind patterns.
- Traffic Data: Collected from available sources.
- Health Data: Hospital admissions for respiratory conditions.

By integrating these datasets, the goal is to develop an AI-driven system that can forecast high-pollution days and support public health strategies.



 Features
Automated data collection from APIs  
Real-time updates with an **Agentic AI** system  
Predictive machine learning models for pollution forecasting  
Data visualization for better insights
